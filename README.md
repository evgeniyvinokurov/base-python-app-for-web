# base-python-app-for-web  2.0.5  
  
Bottle база для проектов на bottle, так же генерированный статичный сайт в шаблонах на основе других проектов и примеры демо проектов.  

файлы:  
 - projects/** - python код проектов  
 - views/** - вью формы питон проектов на bottle  
 - static/** - фронтенд код, стили и остальная статика
